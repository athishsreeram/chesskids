# ♟ ChessKids — Interactive Chess Learning App

> A fun, toddler-friendly chess learning app with 10 mini-games and a full structured curriculum for young players aged 5–14.

-----

## 🚀 Live App

**<https://athishsreeram.github.io/chesskids/>**

-----

## 📱 What Is ChessKids?

ChessKids is a **single-file, offline-capable** chess learning web app designed for young players. It combines interactive mini-games with a structured chess curriculum to make learning chess fun, colourful, and engaging.

No downloads. No logins. No app store. Just open in a browser on any phone or tablet and play.

-----

## 🎮 Mini-Games (10 Games)

|Game          |What It Teaches                                |
|--------------|-----------------------------------------------|
|🎯 Memory Game |Remember piece positions on the board          |
|👀 Blindfold   |Visualise squares without seeing the board     |
|♟ Piece Finder|Find all legal moves for a piece               |
|🧩 Patterns    |Identify tactical patterns (Fork, Pin, Skewer…)|
|🌳 Openings    |Explore common chess openings interactively    |
|👑 Endgame Fun |Practice winning endgame positions             |
|🔎 Board Vision|Identify all squares a piece attacks           |
|🧠 Calculate!  |Mental calculation and piece movement          |
|✅ Think Smart |Learn the structured thinking process          |
|⚡ Focus Flash |Memory and concentration training              |

Each game has 3 difficulty levels: ⭐ Beginner · ⭐⭐ Explorer · ⭐⭐⭐ Hero

## 📚 Curriculum (Structured Learning Path)

Access the full curriculum via the **📚 Courses** tab in the app.

### Level Overview

|Level|Name                                      |Duration                    |Goal                                  |
|-----|------------------------------------------|----------------------------|--------------------------------------|
|L1 🟢 |Beginner — Foundations                    |~2.5 months / 20 sessions   |Play a complete legal chess game      |
|L2 🔵 |Advanced Beginner — Thinking Before Moving|~2.5 months / 20 sessions   |Spot simple tactics and avoid blunders|
|L3 🟣 |Intermediate — Chess Strategy             |~2.5 months / 20 sessions   |Plan 3–4 moves ahead                  |
|L4 🟠 |Advanced — Tournament Thinking            |~2.5 months / 20 sessions   |Play structured tournament-style games|
|L5 🔴 |Advanced Part 2 — Competitive Player      |~2.5 months / 20 sessions   |Compete in local tournaments          |
|★ 🏆  |Master Course — Performance Track         |Sub Junior / Junior / Senior|Rated tournament performance          |

### What Each Session Contains

Every session includes:

- **📖 Concept Card** — Simple, illustrated explanation in kid-friendly language with a real chess board showing the idea
- **🧩 Spot It Puzzle** — Tap the correct square on a live interactive board
- **✅ Quick Quiz** — Multiple choice questions with instant feedback and celebration

### Weekly Class Structure

Each class is **45 minutes**:

|Time  |Activity              |
|------|----------------------|
|10 min|📖 Concept introduction|
|15 min|🧩 Puzzle solving      |
|15 min|♟ Guided gameplay     |
|5 min |🔍 Game review         |

### Training Components (Every Level)

- Concept Training
- Tactical Puzzle Practice
- Guided Chess Games
- Game Analysis
- Mini Tournament Practice
